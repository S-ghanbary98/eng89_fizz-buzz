#FIZZ-BUZZ Game

- This is a game where the multiple of 3 in a range represent the word "FIZZ",
and the multiples of 5 represent the word "BUZZ".
  
```python
for i in range(100):
    if i % 3 == 0 and i % 5 == 0:
        print("FIZZ BUZZ")
    elif i % 5 == 0:
        print("BUZZ")
    elif i % 3 == 0:
        print("FIZZ")
    else:
        print(i)

```
